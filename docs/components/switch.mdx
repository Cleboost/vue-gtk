## Overview

The Switch component is a toggle switch that provides a clean and intuitive way to toggle between on/off states. It follows GTK design principles and supports Vue's v-model for two-way data binding.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `'small' \| 'medium' \| 'large'` | `'medium'` | Defines the size of the switch |
| `modelValue` | `boolean` | `true` | The checked state of the switch (use with v-model) |

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `update:modelValue` | `boolean` | Emitted when the switch state changes. Use with v-model for two-way binding |

## Basic Usage

The Switch component uses Vue's v-model for two-way data binding:

```vue
<template>
  <div>
    <Switch v-model="isEnabled" />
    <p>Switch is: {{ isEnabled ? 'ON' : 'OFF' }}</p>
  </div>
</template>

<script setup>
import { ref } from 'vue';
import { Switch } from 'vue-gtk';

const isEnabled = ref(true);
</script>
```

## Sizes

The Switch component supports three sizes: `small`, `medium`, and `large`.

```vue
<template>
  <div class="switch-demo">
    <Switch v-model="smallSwitch" size="small" />
    <Switch v-model="mediumSwitch" size="medium" />
    <Switch v-model="largeSwitch" size="large" />
  </div>
</template>

<script setup>
import { ref } from 'vue';
import { Switch } from 'vue-gtk';

const smallSwitch = ref(true);
const mediumSwitch = ref(true);
const largeSwitch = ref(true);
</script>

<style scoped>
.switch-demo {
  display: flex;
  gap: 16px;
  align-items: center;
}
</style>
```

### Size Specifications

- **Small**: 28px × 16px with a 12px thumb
- **Medium**: 36px × 20px with a 16px thumb (default)
- **Large**: 44px × 24px with a 20px thumb

## States

### Checked (ON)
When the switch is checked, it displays a blue background (`#4285f4`) with a white thumb positioned on the right side.

### Unchecked (OFF)
When unchecked, the switch displays a gray background (`#53555D`) with a light gray thumb (`#CCCED3`) positioned on the left side.

## Two-way Data Binding

The Switch component fully supports Vue's v-model directive:

```vue
<template>
  <Switch v-model="myValue" />
</template>

<script setup>
import { ref } from 'vue';
import { Switch } from 'vue-gtk';

const myValue = ref(false);
</script>
```

You can also use the explicit syntax if preferred:

```vue
<template>
  <Switch 
    :model-value="myValue" 
    @update:model-value="myValue = $event" 
  />
</template>
```
